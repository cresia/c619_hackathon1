Oh No! Zombies!

Lives

3

Roll Dice to Start the Game

Game Instructions:

1. Roll a Dice to start the game for hidden Zombies on the board

2. Move the player to get a battery at every house while avoiding the hidden Zombies

3. If the player encounters a zombies, the player will lose one Life, The player cannot touch the zombie multiple times to avoid losing the lives

4. The player will lose the game if they lose all of the lives

5. The player will win the game if the player is able to get all of the batteries and still have at least one life left to return to the start point.
